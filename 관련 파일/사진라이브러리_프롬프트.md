# 사진 라이브러리 수집기 구현 지시서

⚠️ 이 프로그램은 두 곳에서 나눠서 작업해야 합니다:
- 대시보드 UI (사이드바 메뉴, 입력폼, 결과 표시) → Claude Code에서 구현
- 이미지 크롤링 (샤오홍슈/바이두 브라우저 자동화) → 안티그래비티 브라우저 에이전트에서 처리

---

## 1. 목표

샤오홍슈(小红书) + 바이두(百度) 이미지에서 제품 카테고리 관련 사진을 대량 수집하고, 모자이크 처리 후 블로그/카페 원고의 [이미지] 위치에 매칭할 수 있는 사진 라이브러리를 구축한다.

AI 이미지 생성이 아니라 실제 사진 수집 방식.

---

## 2. UI 구성

대시보드 "사진 라이브러리" 메뉴:

### 입력 영역
1. **검색 키워드**: 한국어로 입력 (예: "전립선 영양제 복용")
   - 중국어 자동 번역 표시 (번역 결과 확인 후 수정 가능)
2. **수집 소스 선택**: 체크박스
   - ☑ 샤오홍슈
   - ☑ 바이두 이미지
3. **수집 개수**: 숫자 입력 (기본 50장)
4. **카테고리 태그**: 드롭다운 복수 선택
   - 제품 사진 / 복용 사진 / 일상 사진 / 성분 사진 / 효과 사진 / 직접입력
5. **"수집 시작" 버튼** + 프로그레스바

### 결과 영역
- 수집된 이미지 그리드 (썸네일 형태)
- 각 이미지마다:
  - 카테고리 태그 드롭다운 (수동 변경 가능)
  - 삭제 버튼 (안 쓸 이미지 제거)
  - 원본 크게 보기
- 상단 버튼:
  - "모자이크 처리" — 선택한 이미지의 텍스트/로고/얼굴 영역 자동 모자이크
  - "전체 모자이크" — 수집된 전체 이미지 일괄 모자이크
  - "라이브러리 저장" — 로컬 폴더에 카테고리별로 저장
  - "엑셀 목록 다운로드" — 이미지 파일명 + 카테고리 태그 목록

### 라이브러리 탭
- 기존에 저장한 이미지 라이브러리 전체 열람
- 카테고리별 필터
- 키워드 검색
- 이미지 선택 → "복사" (블로그/카페 원고에서 사용)

---

## 3. 백엔드 로직

### 3-1. 키워드 번역
한국어 → 중국어 번역. Google Translate API 또는 내장 번역 사용.
예: "전립선 영양제" → "前列腺保健品"

### 3-2. 이미지 수집 (브라우저 에이전트)

**샤오홍슈:**
1. 브라우저 에이전트가 샤오홍슈 접속 (로그인 필요할 수 있음)
2. 번역된 키워드로 검색
3. 게시물 목록에서 이미지 URL 추출
4. 고해상도 이미지 다운로드
5. 지정된 수집 개수만큼 반복

**바이두 이미지:**
1. 브라우저 에이전트가 https://image.baidu.com 접속
2. 번역된 키워드로 검색
3. 무한 스크롤로 이미지 로딩
4. 고해상도 원본 이미지 URL 추출 + 다운로드
5. 지정된 수집 개수만큼 반복

### 3-3. 모자이크 처리
Python Pillow 라이브러리로 처리:
- 이미지에서 텍스트 영역 감지 → 블러 처리
- 로고/워터마크 영역 감지 → 블러 처리
- 얼굴 영역 감지 → 블러 처리 (개인정보 보호)
- 자동 감지가 어려우면 수동으로 영역 지정 가능

### 3-4. 저장 구조
```
사진라이브러리/
├── 제품사진/
│   ├── 001.jpg
│   ├── 002.jpg
│   └── ...
├── 복용사진/
├── 일상사진/
├── 성분사진/
├── 효과사진/
└── library_index.json  (파일명 + 카테고리 + 키워드 + 수집일)
```

## 4. 블로그/카페 원고와 연동

나중에 블로그/카페 원고 생성 시 [이미지1], [이미지2] 위치에 라이브러리에서 적합한 사진을 자동 매칭하는 기능 추가 가능.
- 원고의 단락 내용을 분석해서 → 카테고리 추정 → 해당 카테고리 폴더에서 랜덤 또는 순차적으로 사진 선택
- 지금은 수집+관리까지만 구현. 자동 매칭은 추후.

---

## 5. 기술 요구사항
- 대시보드 UI는 기존 dashboard.html + server.py에 메뉴 추가
- 이미지 크롤링은 안티그래비티 브라우저 에이전트가 처리 (Selenium/Playwright)
- 모자이크는 Python Pillow (pip install Pillow)
- 이미지 저장은 서버 로컬 폴더
- 대용량 이미지 처리 시 썸네일 생성해서 UI 로딩 속도 확보
- 브라우저 에이전트 사용 시 딜레이 2~3초 (차단 방지)

## 6. 안티그래비티에 시킬 내용 (오른쪽 채팅에 이렇게 말하세요)

"샤오홍슈와 바이두 이미지에서 사진을 수집하는 크롤링 기능을 만들어줘.
- 검색 키워드를 중국어로 번역해서 검색

- 바이두 이미지: 검색 결과 고해상도 이미지 다운로드
- 다운로드한 이미지를 카테고리별 폴더에 저장
- server.py에 /api/collect-images 엔드포인트 추가해서 대시보드와 연결"
