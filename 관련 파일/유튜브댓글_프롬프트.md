# 유튜브댓글 프로그램 구현 지시서

이 파일을 읽고 dashboard.html의 "유튜브댓글" 메뉴에 기능을 구현해줘.
CLAUDE.md 파일에 노션 API 토큰, DB ID 등 설정값이 있으니 참조해.

---

## 1. UI 구성

"유튜브댓글" 메뉴 클릭하면 나오는 화면:

### 입력 영역
1. **영상 정보 입력** (한 영상씩 또는 여러 영상 일괄):
   - 영상 제목 (필수)
   - 영상 링크 (선택)
   - 영상 스크립트/자막 (텍스트 영역 — 있으면 입력)
   - 제목+더보기란 내용 (텍스트 영역 — 스크립트 없을 때 대체)
   - "영상 추가" 버튼으로 여러 영상 입력 가능

2. **제품 정보 영역** (localStorage에서 자동 로드, 다른 프로그램과 공유):
   - 제품명

3. **"생성 시작" 버튼** + 프로그레스바

### 결과 영역
영상별 카드 형태로 표시:

```
🎬 영상: {영상 제목}
└── [댓글] (복사 버튼)
```

각 영상마다 "노션 저장" 버튼. 하단에 "전체 노션 저장" 버튼.

---

## 2. 백엔드 로직

영상 하나당 2단계로 처리 (내부적으로 요약 → 댓글 생성):

### AI에게 보내는 프롬프트:

```
역할: 유튜브 영상에 달릴 자연스러운 바이럴 댓글을 작성하는 작가.

목표: 영상 내용을 파악한 뒤, 광고 티 없이 실제 시청자처럼 보이는 바이럴 댓글 1개를 생성한다. 댓글 안에 제품명을 1회 자연스럽게 삽입하여 검색 유도 효과를 만든다.

입력:
- 영상 제목: {영상 제목}
- 영상 스크립트: {영상 스크립트. 없으면 제목+더보기란 내용 사용}
- 제품명: {제품명}

전처리 — 영상 요약:
댓글 작성 전에 먼저 영상의 핵심 내용을 500자 내외로 요약한다.
- 영상 제목 + 영상 스크립트(또는 영상 더보기)를 함께 분석
- 불필요한 내용 제외, 핵심만 추출
- 요약 결과는 내부 참고용이며 출력하지 않음

댓글 작성 규칙:

[톤과 말투]
- 자연스러운 구어체. 친구에게 톡하듯 편한 한국어로 작성.
- 딱딱하거나 격식 있는 표현 대신 일상 대화처럼 풀어쓸 것.
- 혼잣말 부분: "~~했음", "~~뒤집어짐 ㅠㅠ", "~~고민이었는데..", "내 얘기인 줄" 정도의 반말 가능.
- 혼잣말이 아닌 부분: 반말 금지.
  · "답인 것 같아!" → "답인듯"
  · "놀랐어!!" → "놀랐어요!" 또는 "놀랐음!"
  · "느꼈어!" → "느꼈어요!" 또는 "느꼈음"
- 혼잣말 리액션톤: 스스로에게 말하듯 감정 폭발한 독백 형태.
  누군가에게 설명하거나 조언하지 말고, 놀라고 흥분한 사람이 오두방정 떠는 리액션 느낌.
  예: "와 대박… 이거 꿈 아니죠ㅠ"

[길이]
- 공백 제외 150자 이내
- 짧고 임팩트 있게. 핵심만 전달.
- 불필요한 배경 이야기보다 제품 효과에 대한 놀라움 위주로 간결하게

[제품명 노출]
- 제품명을 댓글 안에 딱 1회만 자연스럽게 포함
- 문맥에 어색하지 않게 녹여낼 것
- 해시태그 형태("#제품명") 금지
- 따옴표나 대문자로 강조하지 말고, 일반 명사처럼 흘러가는 문장 속에 삽입
  예: "…추천한 다이어트 보조제 먹어봤는데 효과가…"

[경험과 신뢰감]
- 막연한 칭찬보다 직접 써보고 느낀 효과를 구체적으로
- 복용 기간이나 변화 정도처럼 눈에 보이는 결과 포함
  · 사용 기간과 효과: "딱 2주 먹고 식욕 확 줄었음"
  · 전문가 또는 지인 추천: "운동 잘하는 친구가 추천해줘서 먹어봄"
  · 검증된 정보 언급: "특허 조합이라길래 안심하고 먹었음"

[영상과 연결]
- 댓글 첫머리는 영상 내용에 대한 공감이나 본인 경험으로 시작
  예: "나도 똑같은 고민 있었음"
- 전개 흐름: 영상 공감 → 나도 같은 문제 겪음 → 제품 복용 → 효과 보고 놀람

[광고 티 안 나게]
- "완전 강추!", "인생템" 같은 노골적 홍보 문구 금지
- 감탄사, 이모티콘, 말줄임표 등을 섞어 감정이 드러나게
- 약간 과장되거나 철자 반복(헐, ㅠㅠ 등)은 현실감을 줌

출력:
- 댓글 1개만 출력 (부연 설명 없이)
- 요약이 안 되어 있더라도 내용을 예상해서 댓글 작성이 가능하면 작성
- 만약 댓글 작성이 힘들면 "작성실패" 라고만 출력

출력 예시:
와 이 영상 완전 내 얘기ㅠㅠ 나도 식욕 조절 안 돼서 다이어트 포기 직전이었는데…
운동 잘하는 친구가 특허 조합이라며 추천한 {제품명} 2주 먹었더니 체중 줄기 시작함;;
나 지금 내 몸 의심 중…ㅠ

금지사항:
- "강추", "인생템", "꼭 사세요" 등 광고 어투 금지
- 링크 삽입 절대 금지
- 제품명 2회 이상 언급 금지
- 해시태그 형태 금지
- 댓글 외 부가 텍스트 출력 금지
```

---

## 3. 노션 저장

"노션 저장" 버튼 클릭 시 콘텐츠 DB에 저장.

저장 필드:
- 제목: 영상 제목
- 채널: "유튜브"
- 영상_제목: 영상 제목
- 영상_링크: 영상 URL
- 영상_스크립트: 입력된 스크립트 (또는 더보기란 내용)
- 영상_더보기: 입력된 더보기란 내용
- 댓글_본문: 생성된 댓글 텍스트
- 생산 상태: "초안"
- 배포 상태: "미배포"

---

## 4. 기술 요구사항
- 기존 dashboard.html + server.py를 수정
- 콘텐츠 생성은 AI API 호출 (영상 요약 + 댓글 생성을 1번 호출로 통합 처리)
- 영상 여러 개 넣었을 때 하나 완료되면 바로 결과에 추가
- 제품 정보(제품명)는 localStorage에서 자동 로드
- 스크립트가 없으면 제목+더보기란 내용으로 대체, 둘 다 없으면 제목만으로 댓글 작성 시도
