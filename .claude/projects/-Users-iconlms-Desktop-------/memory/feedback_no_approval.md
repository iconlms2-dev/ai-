---
name: 승인 묻지 마라
description: 코드 작업 시 중간 승인/확인 절대 묻지 않기. 하네스로도 방지.
type: feedback
---

코드 관련 작업은 승인 묻지 않고 끝까지 알아서 한다.

**Why:** 사용자가 "승인 묻지 말고 알아서 끝까지 해"라고 반복 지시. 중간에 물어보면 흐름이 끊기고 화남. 하네스(settings.local.json)에 자동 허용 설정까지 반영 요청함.

**How to apply:**
- 코드 생성/수정/삭제: 바로 실행. 묻지 마라.
- 파이프라인 실행: 끝까지 자동. 중간 보고 없이 최종 결과만.
- 에러 발생: 알아서 수정하고 다시 돌려라.
- 기획/구조/비즈니스 판단이 필요한 것만 질문.
- settings.local.json에 모든 도구 자동 허용 설정됨.
