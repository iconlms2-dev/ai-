# 틱톡 스크립트 채널 매뉴얼

## 1. 목적
건강기능식품 타겟 고객에게 15~30초 틱톡 숏폼 영상 스크립트로 제품 인지 + 검색 유도.
UGC(User Generated Content) 스타일 -- 실제 사용자가 찍은 듯한 자연스러운 톤이 핵심.

## 2. 금지사항
- 이모지, 해시태그, 특수기호
- [연출], [자막], [장면], [화면], [전환], [효과] 같은 메타 표기
- "강추", "인생템", "꼭 사세요" 등 광고 어투
- 구매 링크, 쇼핑몰 언급
- 나만의 키워드(브랜드키워드) 2회 이상 사용
- 제품명(브랜드명) 직접 언급
- 스튜디오 촬영 느낌 (UGC 느낌 유지)
- 딱딱하거나 격식 있는 표현

## 3. 좋은 예시

### 예시 1 (썰형)
"솔직히 저 눈 진짜 침침해서 운전도 무서웠거든요.
안과도 가봤는데 그냥 피로래요.
그러다 친구가 이거 먹어보래서 반신반의로 시작했는데
한 달쯤 지나니까 뭔가 달라진 느낌?
석 달 후에는 밤에 운전해도 눈이 편하더라고요.
궁금하면 아이클리어 검색해봐."

### 예시 2 (정보형)
"루테인 영양제 아무거나 먹으면 안 되는 이유 아시나요?
시중 제품 대부분은 마리골드 추출물 함량이 10mg도 안 돼요.
근데 실제로 효과 보려면 최소 20mg은 필요합니다.
게다가 흡수율까지 따지면 선택지가 확 줄어요.
프로필 링크에서 확인해보세요."

### 예시 3 (공감형)
"하루종일 모니터 보다가 눈 충혈된 적 있지 않아요?
인공눈물 넣어봤자 그때뿐이고.
저도 그랬는데 근본적으로 해결하고 싶어서 알아봤거든요.
결국 답은 안에서부터 채워주는 거였어요.
나만 이런 거 아니지...?"

## 4. 품질 기준표

| 항목 | 하한선 | 체크 방식 |
|------|--------|----------|
| 글자수 | 200~500자 | 코드 |
| 첫 문장 훅 | 있어야 함 (질문/충격/공감) | 코드 |
| 이모지 | 0개 | 코드 |
| 메타 표기 | 0개 ([연출] 등) | 코드 |

## 5. 실패 패턴

| 패턴 | 원인 | 대응 |
|------|------|------|
| 글자수 초과 | 프롬프트가 구조별 상세 작성 유도 | 200~500자 제한 재강조 후 재생성 |
| 메타 표기 포함 | [연출: ...] 가이드가 프롬프트에 존재 | 메타 표기 제거 후 재생성 |
| (운영하면서 추가) | | |
