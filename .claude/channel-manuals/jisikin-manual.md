# 지식인 채널 매뉴얼

## 목적
네이버 지식인 자연 노출. 실제 사용자가 작성한 것처럼 보이는 Q&A 콘텐츠로 검색 유입 + 제품 인지도 확보. 질문과 답변을 각각 별도 계정으로 올려 자연스러운 상담 형태를 만든다.

## 금지
- 노골적 광고 답변 ("이 제품 사세요", "강추합니다", "대박 효과")
- 링크 남발 (구매 링크, 블로그 링크 직접 삽입)
- 의학적 효능 단정 ("~을 치료합니다", "~에 효과가 확실합니다")
- 동일 패턴 반복 (같은 질문 구조를 키워드만 바꿔 복붙)
- 과도한 전문 용어 나열 (일반 소비자가 쓰지 않을 표현)
- 답변에서 브랜드명 직접 언급 (자연스러운 경험담 형태로만)
- 질문과 답변이 동일 톤/문체 (다른 사람이 쓴 것처럼 보여야 함)

## 좋은 예시

### 예시 1 (고민 상담형)
질문: "요즘 눈이 너무 뻑뻑하고 충혈이 자주 되는데 루테인 먹으면 좀 나아질까요? 40대인데 주변에서 먹으라고 하는데 효과 있는 분?"
답변: "저도 작년에 비슷한 증상이었는데, 안과에서는 건조증이라고만 하더라고요. 루테인을 알아보니까 마리골드에서 추출한 게 흡수가 잘 된다고 해서, 함량 20mg 이상인 걸로 골랐어요. 한 두 달쯤 꾸준히 먹으니까 뻑뻑한 게 확실히 줄었습니다. 지아잔틴도 같이 들어있는 게 좋다고 해요."

### 예시 2 (비교 질문형)
질문: "루테인 영양제 고르려는데 마리골드 추출이랑 합성이랑 차이가 큰가요? 어떤 기준으로 골라야 하나요"
답변: "마리골드 꽃에서 추출한 루테인이 체내 흡수율이 높다는 연구 결과가 있어요. 합성은 가격이 싸긴 한데, 장기 복용하면 차이가 난다는 후기가 많더라고요. 저는 함량 20mg 이상, 지아잔틴 포함 여부, 부형제 종류 이 세 가지 기준으로 골랐습니다."

### 예시 3 (경험 공유형)
질문: "직장인인데 하루종일 모니터 보면서 일하다보니 눈이 침침해요. 루테인 말고 다른 좋은 방법 있나요?"
답변: "모니터 블루라이트 차단 안경이랑 20-20-20 규칙(20분마다 20초간 20피트 멀리 보기)도 같이 하면 좋아요. 근데 저도 그것만으로는 부족해서 루테인을 추가했는데, 마리골드 추출 20mg짜리로 3개월째 먹고 있거든요. 침침한 게 확실히 나아졌어요. 영양제는 꾸준히가 핵심이라 최소 2개월은 드셔보세요."

## 품질 기준표

| 항목 | 하한선 | 체크 방식 |
|------|--------|----------|
| 답변 글자수 | 300자 이상 (답변1 기준) | 코드 |
| Q/A 분리 | 질문과 답변이 다른 내용 | 코드 |
| 키워드 포함 | 제목 또는 답변에 키워드 존재 | 코드 |
| 질문 자연스러움 | 실제 고민처럼 보이는 톤 | AI |
| 답변 전문성 | 경험 기반 + 근거 제시 | AI |
| 광고성 배제 | 노골적 제품 홍보 없음 | 코드 + AI |

## 실패 패턴

| 패턴 | 원인 | 대응 |
|------|------|------|
| 답변이 너무 짧음 (<300자) | 프롬프트 지시 미흡 | 재생성 (최대 3회) |
| 질문-답변 톤 동일 | 같은 페르소나 | 프롬프트에서 페르소나 분리 강화 |
| 브랜드명 직접 노출 | 프롬프트 제어 실패 | 후처리로 브랜드명 제거/교체 |
| 광고성 표현 포함 | 금지어 목록 누락 | rule_validate에 금지어 추가 |
| 키워드 누락 | 답변이 키워드와 무관 | 재생성 트리거 |
