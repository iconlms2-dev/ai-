# 유튜브 댓글 채널 매뉴얼

## 목적
유튜브 영상 댓글을 통해 브랜드/제품을 자연스럽게 노출하여 검색 유도 및 인지도를 높인다.
3단 시나리오(밑밥-해결사-쐐기) 조로 실제 시청자 간 대화처럼 보이게 하여 광고 거부감을 최소화한다.

## 금지
- 링크 직접 삽입 (URL, 단축링크, 프로필 링크 등)
- 브랜드명/제품명 직접 언급 (검색 유도 키워드로 우회)
- 스팸성 표현 ("지금 바로", "할인", "최저가", "쿠폰", "이벤트")
- 봇 느낌의 기계적 문체 (동일 패턴 반복, 정형화된 문장)
- 과도한 이모지 사용
- 동일 영상에 3단 시나리오 동시 게시 (시간차 필요)
- 영상 내용과 무관한 댓글 작성

## 좋은 예시

### 예시 1 (루테인 관련 영상)
- 밑밥: "저도 요즘 눈이 너무 침침한데 루테인이 진짜 효과 있나요? 종류가 너무 많아서 뭘 골라야 할지 모르겠어요ㅠ"
- 해결사: "저는 작년부터 먹고 있는데 확실히 달라졌어요! 핵심은 함량이랑 흡수율인데 아이클리어 한번 검색해보세요 성분 비교 잘 되어있더라고요"
- 쐐기: "오 저도 찾아봐야겠다 요즘 모니터 많이 보니까 눈 관리 진짜 중요한 것 같아요"

### 예시 2 (다이어트 관련 영상)
- 밑밥: "운동은 꾸준히 하는데 체중이 안 빠져서 고민이에요 혹시 식이섬유 보충제 드시는 분 계세요?"
- 해결사: "저는 장 건강부터 잡으니까 오히려 살이 빠지더라고요 장클린이라고 검색해보시면 후기 많아요"
- 쐐기: "헐 장 건강이랑 다이어트가 연관이 있구나 나도 한번 알아봐야겠다"

### 예시 3 (수면 관련 영상)
- 밑밥: "잠을 자도 개운하지가 않은데 저만 이런가요? 수면 영양제 효과 있으신 분?"
- 해결사: "마그네슘이 핵심인데요 흡수율 좋은 거 찾으시면 잠드림이라고 한번 보세요 저는 2주차부터 달라졌어요"
- 쐐기: "마그네슘이 수면에 그렇게 중요한 줄 몰랐네 나도 한번 찾아봐야지"

## 품질 기준표

| 항목 | 기준 | 체크 방식 |
|------|------|----------|
| 글자수 | 50~200자 (댓글 1개당) | 코드 |
| 영상 관련성 | 영상 제목의 핵심 단어 포함 | 코드 |
| 자연스러움 | 실제 유튜브 댓글 톤 유지 | AI |
| 링크 미포함 | URL/링크 패턴 없음 | 코드 |
| 광고성 미포함 | 스팸 단어 없음 | 코드 |
| 3단 구조 | 밑밥-해결사-쐐기 순서 유지 | AI |
| 브랜드 우회 | 직접 언급 대신 검색 유도 | AI |

## 실패 패턴

| 패턴 | 원인 | 대응 |
|------|------|------|
| 글자수 50자 미만 | 댓글이 너무 짧음 | 재생성 시 최소 50자 지시 |
| 글자수 200자 초과 | 댓글이 너무 김 | 재생성 시 200자 이내 지시 |
| 영상 무관 댓글 | 프롬프트에 영상 정보 부족 | 영상 설명 추가 크롤링 |
| 광고 티 과다 | 브랜드명 직접 노출 | 프롬프트 톤 조정 |
| 3단 구조 깨짐 | 파싱 오류 또는 생성 불량 | 파싱 로직 보완 + 재생성 |
