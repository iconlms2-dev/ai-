# 쓰레드 채널 매뉴얼

## 목적
Meta Threads 플랫폼에서 브랜드 노출 및 자연스러운 제품 인지를 유도한다.
일상글(daily)로 페르소나를 구축하고, 물길글(traffic)로 검색 키워드와 제품을 자연스럽게 연결한다.

## 금지
- 과도한 광고성 표현 (최저가, 할인, 구매링크, 무료배송, 선착순 등)
- 링크 스팸 (본문 내 URL 직접 삽입)
- 제품명 3회 이상 반복
- 존댓말/반말 혼용 (하나의 톤으로 통일)
- 경쟁사 직접 비하
- 허위/과장 효능 주장 (의약품적 표현 금지)

## 좋은 예시

### 예시 1 (일상글)
"요즘 아침에 일어나면 눈이 뻑뻑한데 나만 그런 거 아니지?
모니터 앞에 8시간 앉아있으면 당연한 건가 싶다가도
이대로 두면 안 될 것 같아서 이것저것 찾아보는 중
같은 고민 있는 사람 있으면 댓글로 알려줘"

### 예시 2 (물길글 - 정보형)
"루테인이 눈 건강에 좋다는 건 알겠는데
진짜 중요한 건 마리골드 꽃에서 추출했느냐래
합성 루테인이랑 체내 흡수율이 다르다고
나도 최근에 알고 바꿨는데 확실히 다른 느낌"

### 예시 3 (물길글 - 공감형)
"40대 넘어서 눈 침침해진 거 실감하는 순간이 있어
운전할 때 표지판이 늦게 보이기 시작하면 진짜 당황스러움
그래서 주변에서 루테인 먹으라고 하길래 시작했는데
한 달쯤 지나니까 좀 나아진 느낌이라 계속 먹는 중"

## 품질 기준표

| 항목 | 기준 | 체크 방식 |
|------|------|----------|
| 글자수 | 100~500자 | 코드 |
| 이모지 | 허용 (단, 5개 이하) | 코드 |
| 광고성 키워드 | 3개 미만 | 코드 |
| 페르소나 유지 | 말투 통일 | 코드 |
| 자연스러움 | 실제 사용자 글처럼 읽힘 | AI |
| 공감 유도 | 독자가 공감할 포인트 존재 | AI |
| 채널 적합도 | Threads 문화에 맞는 톤 | AI |

## 실패 패턴

| 패턴 | 원인 | 대응 |
|------|------|------|
| 블로그 글 같은 장문 | 프롬프트에 길이 제한 미반영 | 500자 상한 엄수 |
| 노골적 광고 | 셀링 로직이 직접 판매형으로 치우침 | 간접 노출 위주로 재생성 |
| 해시태그 과다 | 해시태그 10개 이상 | 5개 이하로 제한 |
| 경어체/반말 혼용 | 페르소나 톤 미고정 | 한 가지 톤으로 통일 후 재생성 |
