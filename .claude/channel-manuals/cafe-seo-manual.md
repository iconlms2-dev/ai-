# 카페SEO 채널 매뉴얼

## 목적
네이버 카페 경험담으로 자연 유입. 실사용 후기 형태의 중간 길이 콘텐츠로 카페 검색 상위 노출 + 댓글을 통한 자연스러운 제품 인지.

## 금지
- "강추", "대박", "미쳤다", "최고의", "인생템" 등 광고성 과장 표현
- 제품 링크 직접 노출 (검색 유도만 허용)
- 의학적 효능 단정 ("~을 치료합니다", "~에 효과가 있습니다")
- 특수문자 나열 (★☆♥♡ 등)
- 해시태그 남발
- 과도한 제품 찬양 (광고 티가 나면 안 됨)
- "무조건 사야", "안 사면 후회" 등 구매 압박 표현

## 좋은 예시

### 예시 1 (일상 경험담)
"요즘 눈이 자꾸 뻑뻑해져서 안과를 갔는데 특별한 건 없다고 하더라고요. 그래서 주변에서 루테인 먹어보라는 말에 한번 찾아봤어요. 마리골드 추출 함량이 중요하다길래 20mg 이상 되는 걸로 골랐는데, 한 달쯤 지나니까 확실히 달라진 느낌이에요."

### 예시 2 (비교 고민형)
"루테인 영양제 고르면서 세 개 정도 비교해봤는데요. 함량도 다 다르고 부형제도 천차만별이더라고요. 저는 결국 함량 20mg에 지아잔틴까지 들어간 제품으로 정했어요. 가격도 나쁘지 않았고요."

### 예시 3 (정보 공유형)
"카페에서 루테인 관련 글을 보고 저도 정리해봤어요. 고를 때 체크할 포인트가 세 가지 있더라고요. 마리골드 꽃 추출인지, 함량이 20mg 이상인지, 지아잔틴이 같이 들어있는지. 이거 세 개만 보면 크게 실패는 안 하는 것 같아요."

## 품질 기준표

| 항목 | 기준 | 체크 방식 |
|------|------|----------|
| 글자수 | 800~1500자 | 코드 |
| 키워드 횟수 | 3~6회 | 코드 |
| 댓글 수 | 3개 이상 | 코드 |
| 광고성 표현 | 없어야 함 | 코드 (금지어 리스트) |
| 자연스러움 | 7/10 | AI |
| 경험담 톤 | 카페 글 느낌 | AI |

## 실패 패턴
| 패턴 | 원인 | 대응 |
|------|------|------|
| 글자수 800 미만 | 프롬프트가 짧은 글 생성 | 재생성 (최대 3회) |
| 글자수 1500 초과 | 프롬프트가 긴 글 생성 | 재생성 (최대 3회) |
| 키워드 과다 (7회+) | 키워드 스터핑 | 재생성 |
| 광고 톤 글 | 과장 표현 포함 | 재생성 + 금지어 체크 |
| 댓글 부족 | API 응답 불완전 | 재생성 |
