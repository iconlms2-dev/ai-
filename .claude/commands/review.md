---
description: Gemini 교차검증용 리뷰 프롬프트 생성
---

현재 플랜 또는 최근 변경사항을 Gemini Gem에 보낼 리뷰 요청으로 정리합니다:
1. .claude/plans/ 폴더에서 최신 플랜 파일을 읽습니다
2. 플랜이 없으면 최근 git diff를 사용하여 변경사항을 수집합니다
3. 아래 형식으로 리뷰 요청 프롬프트를 생성합니다:
   - 변경 배경/목적
   - 변경 내용 요약
   - 리뷰 요청 사항 (빠진 부분, 리스크, 개선점)
4. 생성된 프롬프트를 클립보드에 복사합니다 (pbcopy 사용)
5. "프롬프트가 클립보드에 복사되었습니다. Gemini Gem에 붙여넣기 하세요." 라고 알려줍니다
