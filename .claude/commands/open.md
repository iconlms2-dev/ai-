이전 세션을 이어서 작업합니다.

## 절차

### 1. SESSION_LOG.md 읽기
- 프로젝트 루트의 `SESSION_LOG.md`에서 **마지막 세션 항목**을 읽는다
- 없으면 "이전 세션 기록이 없습니다. 새로 시작할까요?" 출력 후 종료

### 2. 맥락 복구
마지막 세션의 다음 항목을 정리해서 보여준다:
- **지난 세션 완료 내용** (간단 요약)
- **남은 작업 목록** (우선순위 순)
- **이어가기 지시** (다음에 할 일)

### 3. 출력 형식
```
📋 세션 #N (YYYY-MM-DD) 이어가기

✅ 지난번 완료:
  - ...

📌 남은 작업:
  1. ...
  2. ...
  3. ...

▶️ 이어가기: ...
```

### 4. 바로 시작
이어가기 지시에 적힌 작업을 **바로 시작**한다. 확인 묻지 않는다.

## 규칙
- SESSION_LOG.md만 읽고, 수정하지 않는다
- memory/ 파일도 함께 참고한다
- "뭐 할까요?" 묻지 말고 이어가기 지시대로 바로 실행
