# 멘토 카페SEO(외부형) 전체 워크플로우 정보

## 개요

네이버 카페 외부형 바이럴 마케팅. 검색 → 클릭 → 체류 → 신뢰 → 전환 구조.
핵심: "상위글 구조를 분석하고, 그 패턴을 리라이팅하여 복제하는 방식"으로 원고 생성.

---

## 전체 7단계 워크플로우

### 1단계. 상위노출 글 정보 수집 (반자동)

파이썬 스크립트 3개를 순서대로 실행:

1. `cafe_scraper.py`: 키워드 검색 → 카페탭 1위 글의 제목+링크 추출 (input.xlsx → output.xlsx)
2. `cafe_scraper_second.py`: 1위가 광고일 때 2위 글 추출 (input.xlsx → output_second.xlsx)
3. `cafe_analyzer.py`: 추출된 링크의 본문, 글자수, 키워드반복횟수, 사진수 분석 (output.xlsx → analyzed_output.xlsx)

분석 결과를 노션 DB에 수동 입력. 입력 필드:
- 상위노출 키워드 (v)
- 원글 제목 (v)
- 원글 본문 (v or 빈칸)
- 글자수 (공백제외)
- 메인키워드 반복횟수
- 원글 규칙 (v) — 서브키워드, 금칙어 등
- 사진 수
- 나만의 키워드 (v) — 자사 제품명 또는 간접 언급 키워드
- 제품 정보

---

### 2단계. MAKE 시나리오 자동 실행 (자동)

Make.com이 노션 DB를 읽고 Claude에게 제목→본문→댓글을 순서대로 생성시킴.

모듈 흐름:
```
95번(노션 읽기) → 89번(Claude 제목 리라이팅) → 88번(Claude 본문) → 92번(Claude 댓글 10개) → 93번(노션 제목+댓글 저장) → 94번(노션 본문 저장)
```

#### 89번 모듈 (Claude) — 제목 리라이팅
- 입력: 원글 제목 + 메인 키워드
- 역할: 기존 제목의 전체 구조는 유지하고 마지막 말(어미/단어)만 궁금형/공감형으로 변경
- 규칙: 메인 키워드 그대로 유지, 광고 느낌 X, 이모티콘 X, 제목만 출력
- 예시: "다이어트 자전거 도움되나요?" → "다이어트 자전거 효과 있나요?"

#### 88번 모듈 (Claude) — 본문 작성
- 입력: 리라이팅된 제목 + 경쟁사글(원글) + 메인키워드 반복횟수 + 서브키워드 + 글자수 + 금칙어 + 사진수
- 역할: 결핍 고민과 증상 중심의 질문형 글 작성
- 본문 구성 4단계:
  1. 오프닝 — 증상과 문제 얘기, 공감 유도
  2. 기존 해결 실패 경험
  3. 새로운 제품/방법 선택과 고민
  4. 질문형으로 마무리
- 키워드 규칙: 메인키워드 지정횟수+1회, 서브키워드+1회 삽입
- 글자수: 노션 DB 값 ±100자 (공백 제외)
- 사진: [어울릴 사진] + 짧은 설명으로 표기. 지정 사진수+1개 삽입
- 줄바꿈 3~5줄 간격
- 광고 어투 금지, 키워드 누락/금칙어 포함 시 실패
- 출력: 제목 없이 본문만

#### 92번 모듈 (Claude) — 댓글 10개 생성
- 입력: 제목 + 메인키워드 + 나만의 키워드 + 차선책 6가지 분석 데이터
- 역할: 실제 사용자의 다양한 반응처럼 보이는 자연스러운 댓글 10개 생성
- 댓글 흐름:
  - 1~2번: 공감 + 흔히 시도하는 차선책 언급 → 효과 제한적 강조 (제품 언급 X)
  - 3~4번: 해결책 전환 → 건강기능식품 언급, 제품명 없이 신뢰감 제공
  - 5~6번: 제품명 자연스럽게 등장 → 실제 사용 경험담
  - 7번: 체지방 감소 효과 + 제품명 재등장
  - 8~10번: 추가 공감 + 사회적 증거 + 긍정적 마무리
- 차선책 6가지 (댓글 1~2에서 무용론 부각):
  - 자연 식이요법 (꾸준히 어려움, 한계)
  - 기본 영양제/비타민 (특화 기능성 부족)
  - 전통 한약/디톡스 (검증 부족, 고가)
  - 피부과 시술/휴식 (일시적, 비용 부담)
  - 운동/식단 다이어트 (실천 어려움)
  - 일반 다이어트 보조제 (부작용 위험, 요요)
- 각 차선책마다 장점/단점/메시지 포인트가 프롬프트에 포함
- 출력 형식: 한줄에 한 댓글, 줄바꿈 1회만 (2회 금지)
- 각 댓글에 메인 키워드 1개 이상 포함
- 광고 어투 금지

#### 93번 모듈 (노션) — 결과 저장
- 생성된 제목+댓글을 노션 DB에 저장
- 상태를 "make 완료"로 변경

#### 94번 모듈 (노션) — 본문 기록
- 생성된 본문을 노션 DB 항목에 paragraph로 추가 기록

**이 단계 완료 후 노션 상태: "make 완료"**

### 3단계. 사진 첨부 (수동)

- 본문 내 `[어울릴 사진: 설명]` 위치에 실제 이미지 서칭 후 첨부
- 사진 3~5장, 문단 사이마다 1장씩
- 사진 유형: 손 위 제품 컷, 섭취 모습, 체중계, 생활 배경
- 금지: 제품명 확대촬영, 홍보용 배너 이미지
- 첨부 후 `[ ]` 문구 삭제

**완료 후 노션 상태: "사진첨부 완료"**

---

### 4단계. 원고 수정 (수동)

#### 제목 수정
- 직관적+자극적으로 변경
- 나이대/성별/감정 삽입 시 효과적
- 예: "출산탈모 걱정이네요" → "(30대 여자) 출산탈모 때문에 미치겠어요ㅠㅠ"
- 최근 트렌드: 확실한 타겟팅 + 직관적 제목
  - "40대 되면 출산탈모 심해지나요?"
  - "(30대 여) 재생크림 알려주세요"
  - "(43 여자) 발톱무좀 병원 꼭 가야할까요?"

#### 본문 수정
- AI 초안을 사람이 쓴 것처럼 자연스럽게 다듬기
- 타사 제품 언급 → 자사 제품으로 교체
- 나만의 키워드 삽입 (철자 하나도 바꾸면 안 됨)
- 링크 1개를 마지막 문단에 자연스럽게 삽입
- 상위노출 키워드 반복수 유지 (절대 줄이지 말 것)
- 행동 유도: "나만의 키워드 검색" 또는 "링크 클릭"을 유도

#### 금칙어 검사
- 검사 도구: 셀프모아 금칙어 필터링 사이트 (http://blog.word.filter.s3-website.ap-northeast-2.amazonaws.com/)
- 원고 전체를 복붙 → 금지어 위반 목록 확인 → 자연스러운 대체어로 변경
- 자주 나오는 대체 예시:
  - "바르고 자도" → "바르구 자도"
  - "하더라구요" → "했어요"
  - "병원" → "동네"
  - "진료" → "진찰"
  - "가격" → "금액"
- 상위노출 키워드가 금지어에 해당하면 수정하지 않고 진행

#### 댓글 수정
- 1~2번: 공감 + 차선책 제거 (제품 언급 X)
- 3~4번: 해결책 전환 (제품명 없이)
- 5~6번: 자사 제품 자연스럽게 언급+추천
- 7~10번: 추가 공감+추천+마무리
- 차선책 목록 (1~2번에서 무용론 부각):
  다이어트 약물, 운동/식단/앱, 한방 다이어트, 단일 건강기능식품, 카페인 의존, 일반 종합비타민/오메가3/DHA

**완료 후 노션 상태: "규칙검토완료(사진첨부)"**

---

### 5단계. 카페에 게시글 등록 (수동)

#### 로그인
- 메모장에 저장된 아이디/비밀번호 사용
- 비행기 모드 켰다 끈 후 로그인 (IP 변경)
- 브라우저 시크릿모드에서 로그인
- IP보안 설정 반드시 OFF

#### 게시글 작성
- 카페 접속 → '내가 쓴 게시글' → [New] 표시 글 수정 (기존 일상글을 수정하는 방식)
- 노션에서 완성된 원고를 복사 — 메모장을 거쳐서 붙여넣기 (형식 오류 방지)
- 기존 내용 전체 삭제 후 원고 붙여넣기
- 이미지 삽입 + 크기 조절 + 가운데 정렬
- 댓글 허용 체크 → 등록
- 12~24시간 사이에 발행된 글이 상위노출에 유리

---

### 6단계. 댓글 등록 — "광고의 달인" 프로그램 (반자동)

- 좌측 그룹 목록에서 해당 그룹 체크 → 선택 그룹 수정하기
- 상단 "댓글 불러오기" → 메모장 파일 선택
- 댓글을 달 카페 게시글 URL 입력 (카페명/게시글번호 형태)
- 확인 → 실행 → 댓글 10개 자동 등록
- (멘토 메모: "월결제 해야하지만 안티로 직접 만들수 있을듯")

---

### 7단계. 답글 작성 (반자동)

#### GPT 답글 생성
- 커스텀 GPTs "카페 바이럴 답글 생성기" 사용
- 입력 형식:
  ```
  [게시글 내용]
  --
  [댓글1]
  [댓글2]
  [댓글3]
  ...
  ```
  (`--` 구분선 필수)

#### 수동 검수
- 게시글과 싱크(일관성) 확인 — 앞뒤 맞지 않는 답글 수정
- 나만의 키워드가 정확히 들어갔는지 확인 (철자 하나도 바꾸면 안 됨)
- 톤/어투/말투 점검 — 30~40대 여성 커뮤니티 말투
- 이모티콘 1~3개, 3~5문장 이내

#### 최종 마무리
- 카페에 답글 수동 입력
- 게시글 수정에 들어가 댓글 비활성화 (외부 사용자가 "완성된 후기 토론글"로 인식)

**완료 후 노션 상태: "완료"**

## 노션 상태 전이

```
(입력) → make 완료 → 사진첨부 완료 → 규칙검토완료(사진첨부) → 완료
```

## 운영 전략

| 항목 | 규칙 |
|------|------|
| 작성 빈도 | 하루 3~5건, 시간 간격 2~3시간 |
| 계정 관리 | 1계정당 5글 이하, 프로필 실명형 |
| 키워드 구성 | 메인 1개 + 서브 2개 + 변형형 1개 |
| 링크 사용 | 1개 이하, 마지막 문단에 자연스럽게 |
| 사진 첨부 | 3~5장, 생활형 컷 위주 |
| 댓글 세트 | 10개 내외 (공감→후기→조언 순서) |
| 1시간 이내 연속 게시 | 노출률 급감 — 금지 |
| 중복글/이미지 | 검색 제한/저품질 판정 |
| 광고성 과다 문장 | 금지 ("효과 최고!", "무조건 됩니다!" 등) |

## 카페 진입 방법

### 통계약 (카페 운영자와 광고 계약)
- 2025년 기준 네이버 단속 강화로 사실상 불가능

### 아이디 침투 (현실적 방법)
1. 네이버 아이디 여러 개 확보
2. 각 아이디마다 IP/기기/브라우저 분리 (동일 IP 여러 아이디 → 일괄 정지 위험)
3. 타겟 카페 가입 + 등업 (출석/댓글/일상글로 1~2주)
4. 경험담/후기 형태 글 작성 — 링크 없이 나만의 키워드만 자연스럽게
5. 같은 아이디 반복 홍보 금지, 한 카페당 주 1~2회 이하
