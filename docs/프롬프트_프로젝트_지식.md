# 시스템 정보

## 구조
- Claude API를 호출해서 콘텐츠 생성
- 각 채널마다 "시스템 프롬프트"와 "유저 프롬프트" 2개로 나뉨
- 시스템 프롬프트: 역할, 톤, 구조, 규칙 등 고정 지시사항
- 유저 프롬프트: 키워드, 제품 정보 등 매번 바뀌는 입력값

## 채널 목록

네이버 SEO (키워드 기반):
- 블로그 — 네이버 블로그 원고 (제목 + 본문)
- 카페SEO — 네이버 카페 글 (제목 / 본문 / 댓글 각각 별도 프롬프트)
- 지식인 — 네이버 지식인 (질문제목 / 질문본문 / 답변 각각 별도 프롬프트)

SNS:
- 쓰레드 — Meta Threads 포스트 (일상글 / 물길글 3종: 셔플식, 연민식, 후기식 / 댓글)
- 틱톡 — 숏폼 영상 스크립트
- 커뮤니티 — 뽐뿌, DC인사이드 등 침투글

기타:
- 파워컨텐츠 — 네이버 검색광고 랜딩 (광고소재 + 본문)

## 유저 프롬프트 변수

제품 정보 (매번 다른 제품에 적용):
- {제품명} — 판매 제품 이름
- {나만의 키워드} — 검색 유도용 브랜드 키워드
- {USP} — 제품 핵심 특징/차별점
- {타겟층} — 주 구매 대상
- {주요 성분} — 원료/성분

키워드 정보:
- {keyword} — 네이버 검색 키워드
- {구매여정 단계} — 0_무지, 1_인지, 2_호기심, 3_정보습득, 4_의심, 5_구매직전
- {사진수} — 본문 이미지 삽입 개수
- {키워드반복수} — 본문 내 키워드 반복 횟수
